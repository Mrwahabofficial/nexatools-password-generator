What does it do?
----------------
Generates strong, secure passwords instantly!

  - Exclude specific characters if needed
  - Password strength checker built in
  - Save passwords to file securely
  - Works on Windows, Mac and Linux

Requirements
------------
Python 3.x (free download at python.org)
No extra libraries needed — runs out of the box!

How to Run
----------
Step 1: Install Python from python.org if you haven't

Step 2: Extract the ZIP file

Step 3: Open Terminal or Command Prompt

Step 4: Navigate to the folder:
   cd Downloads\NexaTools-Password-Generator\NexaTools-Password-Generator

Step 5: Run the script:
   python password_generator.py

Step 6: Choose option 1 to generate passwords

Step 7: Set your preferences and done!

COMMON ERROR FIX
----------------

Fix: Run this command first:
   cd NexaTools-Password-Generator
Then try again:
